Flashcards

  1. JavaScript (JS): The universal language of web browsers
  2. Operators: Used for operating on strings and intergers
  3. Variables: An entity that can have a value assigned to it
  4. Variable naming conventions: Rules for how to name variables (such as camelCase)
  5. Functions: A block of code designed to perform a specific task; it is executed when something invokes it
  6. Methods: Functions specific to a particular class or id
  7. Arguments: The data passed into a function's parameter
  8. Parameters: The variable in a method definition
  9. Return: The output of a function
  10. Chaining methods: When a method calls another method
  11. Strings: Text elements
  12. Booleans: True/False values
  13. Undefined: a variable has no value assigned to it
  14. NaN: Not a number
  15. Escape: Encodes a string into hex
  16. Alert: Provides a pop-up alert
  17. Comments: Text in code file not used by the program, to make it easier to read by humans
  18. jQuery: A framework of preconfigured Javascript functions
  19. Attributes: A value applied to an element to describe and differentiate it
